1. npm install
2. add .env file in root directory of the project
3. format of env file:

```
# Port and Host address
PORT=
HOSTNAME=

# Api Credentials
CTP_PROJECT_KEY=
CTP_CLIENT_SECRET=
CTP_CLIENT_ID=
CTP_AUTH_URL=
CTP_API_URL=
CTP_SCOPES=
```

4. create commercetool client from merchant center
5. copy the credentials to your .env file
